Library Management System with C#

Access to the system is secured through a login and password, ensuring data privacy and user-specific functionality.

Our dashboard provides a sleek and organized interface, ensuring a user-friendly experience. Here's a glimpse of what it offers:

Our Library Management System features a user-friendly main menu bar that offers easy access to a range of functions. Users can effortlessly select the desired action, including adding new books or viewing the entire catalog. This intuitive design streamlines the user experience, ensuring efficient book management within the system.

"The panel designed for adding a book to the Library Management System offers a user-friendly interface where essential book details can be seamlessly inputted. This includes the book's title, author information, publication details, price, quantity, and purchase date. This meticulous design ensures that all necessary information is efficiently recorded within the system, enabling smooth cataloging and management of the library's resources."

We provide an 'View Book' page that allows users to access and modify book details, providing comprehensive control over the library's catalog. The bottom panel becomes visible upon clicking a book title and disappears when the action is canceled, providing a seamless and user-friendly experience.

Similarly, we can add a student to our system and make any necessary changes to their details at a later time.

The next navigation button, labeled "Issue books," is designed for adding books to the system that students have borrowed from the library. We have set a restriction where each student can borrow a maximum of three books at a time.

The "Return Book" feature allows us to input the student's enrollment number, view the books they have borrowed, and provides the option to return those books to the library.

Finally, we have the "Complete Book Detail" section where we can view a comprehensive list of all borrowed books as well as all returned books in the library.

All data about students and books is securely stored in the SQL Server database, ensuring efficient management and retrieval of information.
